Setelah selesai mengedit, klik Unduh questions.json, lalu upload/commit file itu ke GitHub atau project Vercel Anda.
Format CSV: pertanyaan, opsiA, opsiB, opsiC, opsiD, hurufBenar(a/b/c/d), penjelasan . Baris pertama dianggap header. Setelah impor/tambah soal, klik Unduh questions.json lalu upload ke GitHub/Vercel.
Data soal tidak disimpan di browser. Semua perubahan hanya tersimpan di file questions.json yang Anda unduh & upload ke repo.
